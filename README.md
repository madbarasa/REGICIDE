> **"孤身踏入王座间，筹谋一场弑君之战。"**

《弑君者》（Regicide）是一款极具策略深度的合作纸牌游戏。本项目是该实体桌游的高品质网页数字化移植版，不仅完美还原了原版的“四种花色魔法”、“组合技”、“Boss免疫”等核心机制，更在其基础上独创了**英雄养成系统**与**高智商AI队友**，带来极佳的单人沉浸式卡牌作战体验。

## ✨ 核心特性 / Features

- **🛡️ 完美复刻的战斗系统**：原汁原味的打牌逻辑，支持宠物（Ace）组合技、连招系统（总和 ≤ 10）、与花色技能联动。
  - ♣ 草花 (Clubs)：造成双倍伤害（暴击）。
  - ♠ 黑桃 (Spades)：削减 Boss 的攻击力，作为场上护盾持续生效。
  - ♦ 方块 (Diamonds)：从酒馆（牌库）补充全队手牌（补给）。
  - ♥ 红心 (Hearts)：从墓地（弃牌堆）将卡牌洗回酒馆（复活）。
- **♟️ 12 层堕落君王挑战**：从 J（20血）、Q（30血）一路战至 K（40血），Boss 天生免疫同花色卡牌的特效，只有**精确击杀**（感化）才会让他们加入你的牌库为你效忠。
- **🤖 进阶版协同 AI 系统**：支持单机 1~4 人组队（1真玩+最多3个AI）。AI 具备极强的算力，使用 $O(2^n)$ 状态位掩码引擎进行 0 浪费防守计算，还能自动智能施放其角色的“救命技能”配合你的攻势。
- **🎭 英雄养成机制 (Progression)**：打破传统桌游束缚，引入了角色养成系统。
  - **天选之人 (Chosen One)**：危急时刻拍下 Joker 废除 Boss 被动，洗牌重抽。
  - **吟游诗人 (Bard)**：发动“万能之手”，置换烂牌重塑发牌几率。
  - *角色随胜场累计提升 Level，解锁更强被动天赋。*
- **🪞 沉浸式暗黑美学UI**：深邃的磨砂玻璃态（Glassmorphism） UI 设计，支持自适应各类显示宽度的流式响应式战斗桌面，附带完整的音效碰撞反馈与屏幕抖动视觉特效。

## 🎮 如何游玩 / How to Play

本项目为纯前端静态应用（Vanilla JS + CSS + HTML），**无需任何环境依赖与构建工具**。

1. **直接运行**：使用浏览器直接打开 `index.html` 或者将其部署在任意静态托管服务器（如 GitHub Pages、Vercel 等）中。
2. **操作说明**：
   - 在主界面可选择“冒险模式”来配置你的 AI 队友阵型和扮演的角色。
   - 战斗内直接点击选取你的手牌，系统会自动判定卡牌组合（Combo）的合法性。
   - 若符合要求，点击红色的【发动攻击】对 Boss 倾泻火力！
   - 防守阶段时，屏幕会要求你交出点数 $\ge$ Boss攻击力的手牌（可以交多张），交不出来则游戏失败。
